# PiStats    
Finding and research on environmental statistics using Raspberry PI 3.  
  
#
# Version 1
DS18B20 One-Wire temperature sensor configuration.  
**one-wire support needed** (raspberry pi configuration menu.)
#
## Hardware Required
    * Raspberry Pi
    * DS18B20 Sensor
    * 1x 4,7KΩ Resistor
    * Breadboard
    * Jumper Wires
